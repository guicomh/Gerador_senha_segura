# Gerador de Email e Senhas

Este é um simples gerador de email e senhas em Python.

## Funcionalidades

- **Gerar Senha:** Opção para gerar senhas aleatórias.
- **Gerar Email:** Opção para gerar emails aleatórios no formato `usuario@gmail.com`.
